Description : 

Le programme suivant permet de calculer et afficher graphiquement le chemin le "plus court" possible entre deux sommets d'un graphe:
Il vous propose donc d'ouvrir un fichier parmi trois : graphe1, metro et graphe3. Après la selection de ce fichier vous êtes invité à entrer 
la station de départ et d'arrivée (si vous orthographiez mal le nom une proposition de correction vous sera faite)
Puis le programme vous affiche le chemin à emprunter dans la console, soit avec toutes les stations, soit avec seulement les changements de ligne. 
Enfin une fenêtre s'ouvre avec un affichage graphique du plan du graphe.
Vous pouvez choisir d'afficher l'intégralité du graphe ou seulemnt le parcours choisi, le nom des stations/sommets ou non et zoomer dans la fenêtre.

Installation : 

Pour lancer ce programme, vous avez besoin d'installer deux librairies : SDL_phelma et SDL_ttf.
Ces librairies sont situées par défaut dans le dossier /users/prog1a/C/librairie/2001

Pour lancer le programme, entrez la commande make, puis exécutez le fichier lecture

Description : 

The following program compute and display the "shortest" path possible between two vertices of a graph.

It therefore proposes to open a file among three: graph1, metro and graph3. After selecting this file you are prompted to enter

The station of departure and arrival (if you spell badly the name a proposal of correction will be made to you)
Then the program shows you the path to be taken in the console, either with all the stations or with only the line changes.
Finally, a window opens with a graphic display of the graph plane. You can calculate a new path with double click on two stations of your choice or press R to type it i the console

Installation : 

To compil this program you need to install SDL_phelma and SDL_ttf. You can download SDL_phelma at http://tdinfo.phelma.grenoble-inp.fr/1AS1/graphique/SDL_phelma.zip
Then compil with the make command and start "lecture"  file
